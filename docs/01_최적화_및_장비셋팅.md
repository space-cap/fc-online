# 제1권. 기초와 최적화 (Foundation)

## 01. 최적화 및 장비 셋팅 (Optimization)

FC 온라인은 미세한 반응 속도(Input Lag) 차이가 승패를 가르는 게임입니다.
상대보다 0.1초 먼저 반응하고 싶다면 이 셋팅을 반드시 따라 하세요.

---

### 1. 인게임 그래픽 설정 (Visuals)

"예쁜 화면"이 아니라 **"잘 보이는 화면"**과 **"반응 빠른 화면"**이 목표입니다.

*   **해상도 (Resolution):** 모니터 기본 해상도(Native) 사용. 주사율(Hz)은 모니터 최대치로 설정.
    *   *주의:* 창모드보다 **전체 화면**이 인풋랙이 적습니다.
*   **화면 주사율:** 144Hz 이상 모니터 필수. (60Hz와 144Hz는 수비 반응속도 자체가 다릅니다.)
*   **MSAA (안티앨리어싱):** `끄기` 또는 `2x`. 높일수록 그래픽은 부드러워지나 미세한 랙 유발 가능성.
*   **관중 표시:** `끄기`. 프레임 드랍의 주원인입니다. 랭커들은 모두 끄고 합니다.
*   **경기장 안개/그림자:** `끄기`. 선수 식별에 방해됩니다.

### 2. 엔비디아(NVIDIA) 제어판 최적화 (Advanced)

바탕화면 우클릭 -> NVIDIA 제어판 -> 3D 설정 관리 -> 프로그램 설정 -> FC Online 선택

*   **저지연 모드 (Low Latency Mode):** `울트라(Ultra)` (필수)
    *   마우스/키보드 입력 신호를 CPU가 즉시 처리하여 인풋랙을 최소화합니다.
*   **전원 관리 모드:** `최고 성능 선호`
    *   그래픽카드가 쉬지 않고 풀파워로 돌아가게 합니다.
*   **수직 동기화 (V-Sync):** `끄기`
    *   켜면 화면 찢어짐(Tearing)은 없지만 인풋랙이 심해집니다.

### 3. 키보드 레지스트리 최적화 (Keyboard Response)

키보드 반응 속도를 극한으로 끌어올리는 윈도우 설정입니다.
(`Win + R` -> `regedit` 실행)

> **⚠️ 주의:** 레지스트리 수정은 신중하게 하세요.

1.  경로: `HKEY_CURRENT_USER\Control Panel\Accessibility\Keyboard Response`
2.  값 변경:
    *   **AutoRepeatDelay:** `200` (키를 꾹 눌렀을 때 반복 입력이 시작되는 대기 시간. 짧을수록 빠름)
    *   **AutoRepeatRate:** `6` (반복 입력 속도. 낮을수록 빠름)
    *   **BounceTime:** `0`
    *   **DelayBeforeAcceptance:** `0`
    *   **Flags:** `59` (필터키 끄기 등의 설정)

**설정 후 반드시 재부팅하세요.** 체감이 확 달라집니다.

---

### 4. 카메라 시점 추천 (Camera)

시야가 넓어야 빈 공간이 보입니다.

*   **원거리 (Tele Broadcast):** 가장 무난하고 많이 쓰는 시점. (F4 키)
*   **다이나믹 와이드 (Co-op):** 경기장을 위에서 넓게 내려다보는 시점. 전술적인 움직임을 보기 좋습니다.
    *   *커스텀 설정:* 높이 20 / 줌 0 (최대 시야 확보)

### 5. 키 셋팅의 과학 (Key Mapping)

기본 설정(`Shit` 달리기)보다는 피파 온라인 4/FC 온라인 전용 키 셋팅을 권장합니다.

*   **전력 질주 (Sprint):** `E`
    *   새끼손가락으로 Shift를 누르는 것보다 검/중지로 E를 누르는 것이 반응이 빠릅니다.
*   **선수 변경 (Switch):** `S` (수비 시)
*   **특수 컨트롤 (Modifier):** `C`
*   **개인기 발동:** `Shift`

> **Pro Tip:** 키보드 유저는 '동시 입력'이 되는 게이밍 키보드(무한 동시 입력 지원)가 필수입니다. 3~4키를 동시에 눌러야 하는 상황(예: 대각선 달리기 + 슛 + 방향키)이 많습니다.
