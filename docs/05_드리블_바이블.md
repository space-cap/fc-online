# 제3권. 피지컬과 컨트롤 (Mechanics)

## 05. 드리블 바이블 (Dribbling Bible)

FC 온라인은 개인기를 남발하는 게임이 아닙니다.
적재적소에 딱 하나의 기술로 상대를 제치는 '타이밍'의 예술입니다.

---

### 1. 드리블의 3원칙
1.  **공간:** 빈 공간이 있을 때만 쳐라. 수비수가 앞에 꽉 차 있는데 치면 뺏깁니다.
2.  **심리:** 상대가 '달려들 때' 혹은 '멈칫할 때' 역동작을 걸어라.
3.  **간결함:** 개인기 2번 이상 연속으로 쓰지 마라. (스태미너 낭비 + 수비 복귀 시간 줌)

---

### 2. 별 개수별 필수 개인기 (Skills by Star)

선수의 개인기 등급(별)에 따라 쓸 수 있는 기술이 다릅니다.

#### ⭐⭐⭐ 3성 개인기 (수비수, 투박한 미드필더)
*   **바디 페인팅 (Body Feint):**
    *   커맨드: `Shift + ↑ (유지)` 후 `→` (옆 방향 툭)
    *   가장 기본이지만 가장 강력합니다. 상체를 흔들어 수비를 속입니다.

#### ⭐⭐⭐⭐ 4성 개인기 (대부분의 공격수)
*   **힐투볼 (Heel to Ball Roll):** ★★★★★ (강력 추천)
    *   커맨드: `Shift (유지)` + `↑` (진행방향) 후 `↓` (반대방향) -> (커맨드 수정: `Shift` + `진행방향` 잡고 `옆`(90도) 툭)
    *   *정확한 커맨드:* `Shift`(유지) + `앞`(보고있는 방향) 톡 + `옆`(가고싶은 방향) 톡.
    *   효과: 발뒤꿈치로 공을 툭 치며 옆으로 빠져나갑니다. 박스 안에서 슛 각도 만들 때 최고입니다.
*   **베르바토프 턴 (Berba Spin):**
    *   커맨드: `Shift` + `앞` + `옆` (90도)
    *   사이드에서 중앙으로 꺾어 들어올 때 유용합니다.

#### ⭐⭐⭐⭐⭐ 5성 개인기 (호나우두, 네이마르 등)
*   **스쿱 턴 (Scoop Turn):** ★★★★★ (필수)
    *   커맨드: `A -> S` (슛페이크) + 방향키
    *   효과: 공을 숟가락으로 퍼 올리듯 부드럽게 턴을 합니다.
    *   *발동 조건:* 정지 상태거나 아주 느리게 걷는 상태여야 나갑니다.
*   **엘라스티코 (Elastico):**
    *   커맨드: `Shift` + `오른쪽` -> `반바퀴 돌려서 왼쪽` (반원 그리기)
    *   수비수 다리 사이를 노릴 때 씁니다.

---

### 3. 노터치 드리블과 심리전

키보드 조작만으로 가능한 '고수들의 움직임'입니다.

#### E 퍼스트 터치 ( 치고 달리기)
*   공 받기 전 `E` + 방향키.
*   수비수가 내 등 뒤에 바짝 붙었을 때, 공을 흘리면서 돌아서면 바로 1:1 찬스입니다.

#### C 드리블 (저속 드리블) vs EC 드리블
*   **C 드리블 (`C` 유지):** 몸을 낮추고 공을 지킵니다. 등지기 딱판정에 씁니다.
*   **EC 드리블 (`E`+`C` 유지):** 공을 발에 붙이고 게처럼 움직입니다.
    *   상대 수비가 뺏으려고 `D`를 누르며 달려들 때, EC로 살짝 뒤로 뺐다가 앞으로 가면 수비수는 헛발질을 합니다. 이것이 **'EC 심리전'**의 핵심입니다.

#### Q 부스터 (급정지)
*   달리다가 `Q`를 툭 누르면 즉시 멈춥니다.
*   따라오던 수비수는 관성 때문에 지나쳐버립니다. 그때 다시 달립니다.

---

### 4. 추천 콤보 (Combo)

1.  **사이드 돌파:** 치고 달리기(`E` 퍼스트터치) -> 컷백 각 좁히기 -> `Q` 정지 -> `힐투볼`로 안쪽 진입 -> 패스/슛
2.  **중앙 돌파:** 패스 받음 -> `등지기(C)`로 버팀 -> `스쿱턴(5성)`으로 돌아섬 -> `D` 슈팅

입문자는 **'슛 페이크(AS)'**와 **'힐투볼'** 딱 2개만 연습하세요. 이것만 해도 티어가 바뀝니다.
